# Sample Clean Repo

This folder contains only clean, non-monitored packages.

## Example Commands

```bash
python3 enhanced_npm_compromise_detector_phoenix.py sample_repo_clean --output clean_report.txt
python3 enhanced_npm_compromise_detector_phoenix.py sample_repo_clean --enable-phoenix --output clean_report.txt
python3 -m universal_vulnerability_scanner.main scan sample_repo_clean --output clean_universal_report.txt
```
