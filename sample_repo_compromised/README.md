# Sample Compromised Repo

This folder contains only compromised packages from the SANDWORM_MODE list.

## Example Commands

```bash
python3 enhanced_npm_compromise_detector_phoenix.py sample_repo_compromised --output compromised_report.txt
python3 enhanced_npm_compromise_detector_phoenix.py sample_repo_compromised --enable-phoenix --output compromised_report.txt
python3 -m universal_vulnerability_scanner.main scan sample_repo_compromised --output compromised_universal_report.txt
```
